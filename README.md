# SpendWise 💰
An Android expense tracking application built with Java

## Features
- User authentication with SHA-256 password hashing
- Add, edit, delete daily expenses
- Category-wise expense organization
- Pie chart and bar chart visualization
- SMS auto-detection for bank transactions
- Smart budget alerts at 70%, 90%, 100% threshold
- Daily spending summary notifications

## Tech Stack
Java | Android SDK | Room Database | MVVM | 
LiveData | RecyclerView | MPAndroidChart | 
AlarmManager | BroadcastReceiver
